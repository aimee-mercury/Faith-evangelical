# Church website (React + Vite)

This project recreates the layout and styling from your screenshot as a single-page React site.

## Run locally

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`

## Notes

- This template uses online African photos (Unsplash Source URLs) for backgrounds/avatars.
- To swap images, edit the `background-image` URLs in `src/styles.css`.
